Abstraction is created by using a high level representation to point to a lower level real application software.

Everything in Operating Systems is split between mechanisms and policies
#### Definitions 
- Bare metal: Software running on pure hardware without an operating system
- Virtualization: OS multiplexing allows system resources to be shared between applications
- Kernel: The absolute minimal core software within an OS which provides its core functionalities
- Shell: A text-based command line interpreter that allows users to interact with the OS via a terminal
- System Call: Programmatic interface to the OS. An example of this is a system call operating as a bridge between the shell and the kernel
