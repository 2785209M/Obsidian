If there is an odd number of singularities to the right of an area on the real axis then poles can move into that area on the real axis of the root locus graph. If there is an even number of points to the right, then that section of the graph is closed in the real axis and no poles or zeros can move within it.

The point where the calibration equation equals zero is the start of the root locus

If the root locus is entirely in the left plane then the system is stable for all k>0. If the root locus crosses the y axis the system can be unstable. If the system is entirely in the positive real axis then the system is unstable for all k>0.

Zeros are the values of s for which the numerator equals zero.
Poles are values of s for which the denominator equals zero.

The number of Asymptotes in a root locus plot is given by the number of pole minus the number of zeros. $n-m$. This is known as the relative degree

The angle of each asymptote relative to the positive real axis can be found using the equation $\theta=\frac{180}{n-m}+\frac{k*360}{n-m}$ . To find each asymptote increase K linearly with the number of the asymptote you are calculating.

The Meeting point is given by the equation: $x = \frac{\Sigma R(p)-\Sigma R(z)}{n-m}$ where p and z are the sum of the poles and zeros

If you have $q$ poles at a single point, their angle of departure can be calculated using the equation $\theta_d=\frac{(2k+1)180\degree}{q}$

As K increases, the poles will follow the asymptote.
